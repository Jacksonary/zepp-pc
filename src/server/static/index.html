如何获取 Auth Key？

方法一：使用 huami-token 工具（推荐）

  1. 安装 Python 3.10+ 和 pip
  2. 运行 pip install huami-token
  3. 运行 python -m huami_token
  4. 按提示登录 Zepp 账号
  5. 在输出中找到你的设备对应的 auth_key

方法二：从 Zepp 手机 App 提取

  1. 在手机上安装 Zepp App 并完成手表配对
  2. 使用文件管理器找到 Zepp 缓存目录下的 token 文件
  3. 或在 Android 上使用 logcat 过滤 Zepp 的日志

注意

Auth Key 是 32 位十六进制字符串（16 字节），每个设备的 Key 不同，更换设备后需要重新获取。
